# Go API client for affinity

Affinity API for software platforms connecting practices to the compounder network. A practice is the customer organization, a provider is an individual clinician or prescriber, and a location is a physical practice site. The API covers practice management, catalog discovery, prescription-order submission, fulfillment tracking, and webhooks.

Each operation can use different server URL defined using `OperationServers` map in the `Configuration`.
An operation is uniquely identified by `"{classname}Service.{nickname}"` string.


### affinityApiKey

- **Type**: API key
- **API key parameter name**: x-affinity-api-key
- **Location**: HTTP header

Note, each API key must be added to a map of `map[string]APIKey` where the key is: affinityApiKey and passed in as the auth context for each request.

Example

```go
auth := context.WithValue(
		context.Background(),
		affinity.ContextAPIKeys,
		map[string]affinity.APIKey{
			"affinityApiKey": {Key: "API_KEY_STRING"},
		},
	)
r, err := client.Service.Operation(auth, args)
```
