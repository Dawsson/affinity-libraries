# Quick-start examples

Each example reads `AFFINITY_API_KEY` and performs the safest initial call: inspecting the service
key's organization, mode, and scopes. Use a test-mode key.

- [TypeScript](./typescript/quickstart.ts)
- [Python](./python/quickstart.py)
- [Go](./go/quickstart.go)
- [Java](./java/Quickstart.java)

The generated SDK READMEs contain every endpoint and model. These examples intentionally focus on
the minimal authentication path a first client should run before creating synthetic practices or
orders.
